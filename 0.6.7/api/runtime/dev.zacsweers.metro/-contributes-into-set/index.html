ContributesIntoSet

@Target(allowedTargets = [AnnotationTarget.CLASS])
annotation class ContributesIntoSet(val scope: KClass<*>, val binding: binding<*> = binding<Nothing>(), val replaces: Array<KClass<*>> = [])(source)

Contributes an IntoSet binding of the annotated type to the given scope as a binding (if specified) or single declared supertype.

// Implicit supertype is Base
@ContributesIntoSet(AppScope::class)
@Inject
class Impl : Base

Use binding to specify a specific bound type if an implicit one is not possible.

// Explicit supertype is Base
@ClassKey(Impl::class)
@ContributesIntoSet(AppScope::class, binding = binding<Base>())
@Inject
class Impl : Base, AnotherBase

This annotation is repeatable, allowing for contributions as multiple bound types. Note that all repeated annotations must use the same scope.

If this declaration is scoped, the Scope annotation will be propagated to the generated IntoSet declaration.

If this declaration is qualified, the Qualifier annotation will be propagated to the generated IntoSet declaration.
